Modern JS Calculator
A sleek, responsive, and accessible web-based calculator. This project handles basic arithmetic and scientific operations with a focus on clean UI and error handling.
Features
Basic Arithmetic: Addition, subtraction, multiplication, and division.
Scientific Mode: Includes square roots, percentages, and exponentiation.
Responsive Design: Fully functional on mobile, tablet, and desktop screens.
Keyboard Support: Use your physical numpad to enter calculations.
Theme Toggle: Switch between Light and Dark modes.

Usage
Clear All (AC): Resets the display and memory.
Delete (DEL): Removes the last character entered.
Equals (=): Computes the final result. Supports "Chain Operations" (e.g., 5 + 5 + 5 = 15).
